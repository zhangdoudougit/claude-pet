## 截图建议

- 用 Win+Shift+S 选区截 → 直接粘进编辑器即可
- 1080p 桌面下截图 →直接用,Retina 截图缩到 50% 再放
- 给关键 UI 元素加个红框 / 箭头标注会更友好
