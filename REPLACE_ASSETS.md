# 换图说明 (REPLACE_ASSETS.md)

`assets/` 里现在的是占位图（简陋的 Q 版小娃娃）。你随时可以换成真正的角色图。

# 文件名必须是这 6 个

| 文件名 | 状态 | 触发情境 |
|---|---|---|
| `idle.gif` | 默认 | 没在用 claude |
| `tender.gif` | 温柔 | 偶尔切换的安静状态 |
| `focused.gif` | 专注 | claude 在干活、debug、git |
| `happy.gif` | 活泼 | 搞定/done/完成 |
| `worried.gif` | 担心 | error/报错/失败 |
| `proud.gif` | 得意 | 被夸/谢谢 |

# 规格建议

| 项 | 推荐 | 为什么 |
|---|---|---|
| **格式** | GIF（带循环动画）| 程序自动播放 |
| **格式备选** | PNG（静态）| 程序也能用 PNG，只是没动画 |
| **背景** | 透明 | 程序自己画光晕和气泡 |
| **尺寸** | 200×280 px 或类似比例 | 程序会等比缩放到 180×200 |
| **角色位置** | 居中或偏下，顶部留 20px | 顶部要给"专注态"等状态标签留位置 |
| **大小** | 单个 GIF < 500KB | 加载快，不卡顿 |

# 怎么换

## 方法 1：直接覆盖（推荐）

把你的 GIF 重命名成 `happy.gif` 之类，丢进 `assets/` 文件夹覆盖原文件。

**程序运行中也能覆盖**——文件监听会自动重载，泡沫立刻变成新图。

## 方法 2：批量替换

`assets/` 文件夹删掉，新建一个，把你的 6 张图按规定文件名放进去。重启程序。

# 缺图怎么办？

只换一部分图也行。比如只画了 `happy.gif` 和 `worried.gif`：
- 程序找不到的状态会 **fallback 到 `idle.gif`**
- 都没有的话会显示提示文字"未找到角色图"

# 找/做图的几个思路

## 自己画
- 手绘扫描 → 在线扣背景（remove.bg）→ ezgif.com 转动画 GIF
- iPad + Procreate / Clip Studio → 导出动画 GIF

## AI 生图
- Midjourney / SDXL：搜"Q版立绘 透明背景 紫色头发 女孩"等关键词
- 然后扣背景 + 转 GIF

## 找现成的
- B站/小红书 #VRChat 头像 #立绘 等标签
- 注意版权，自用没事，别公开发

## Live2D 做派生
- 找个 Live2D 模型在 nizima / FaceRig 录屏
- 录 6 段不同表情，每段截 24 帧导出成 GIF

# 进阶：动画时长建议

| 状态 | 推荐 GIF 帧率/时长 | 风格 |
|---|---|---|
| idle | 30 帧 / 3 秒循环 | 慢，呼吸感 |
| tender | 30 帧 / 3 秒循环 | 慢，眨眼 |
| focused | 24 帧 / 2 秒循环 | 几乎不动，偶尔眯眼 |
| happy | 16 帧 / 1.2 秒循环 | 快，跳跃/挥手 |
| worried | 16 帧 / 1.5 秒循环 | 抖动/汗滴 |
| proud | 20 帧 / 2 秒循环 | 微微仰头/星星 |

# 背景颜色注意

如果你的 GIF **不是透明背景**（带白色/纯色底），看起来会有方形底色。要么：
- 重新导出成透明背景
- 或者改 `foamo_pet.py` 里 `WIDGET_W/H` 让窗口刚好等于角色尺寸

# 调整光晕颜色

光晕颜色是程序根据状态生成的（不是图片里的）。改 `foamo_pet.py` 顶部 `STATE_COLORS`：

```python
STATE_COLORS = {
    'happy': ('#ff7eb6', '#c93f7e'),  # 改成你想要的色
    ...
}
```

格式：`('亮色', '暗色')`，亮色用作光晕主色，暗色用作渐变深处。
